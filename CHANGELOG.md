# 更新日志

## [1.0.0] - 2026-03-04

### 新增

- **组件识别与补全**：在 Vue 模板和 WXML 中输入 `<` 自动提示已注册的组件（支持静态 import 和动态 import 两种方式）
- **组件属性补全**：在组件标签内自动提示目标组件的 props / properties，Vue 使用 `:prop=""` 格式，小程序使用 `prop="{{}}}"` 格式
- **Ctrl+Click 跳转定义**：在模板中 Ctrl+点击组件标签名，跳转到组件的 `.vue` 文件或小程序的 `.js` 文件
- **全局组件扫描**：自动扫描项目根目录下 `components/` 和 `component/` 文件夹中的组件并提示
- **多项目工作区支持**：根据当前编辑文件自动定位所属项目根目录，每个项目独立提示全局组件
- **自定义组件文件夹**：通过 `vueWxHelper.componentFolders` 配置自定义全局组件扫描路径
- **Vue data/props/computed/methods 提示**：在 `this.` 后和模板 `{{ }}` / 指令值中提示当前文件定义的属性
- **小程序 data/properties 提示**：在 `this.data.` 后和 WXML `{{ }}` 中提示当前页面/组件定义的属性
- **组件标签高亮**：已注册的组件标签在模板中以绿色（`#4EC9B0`）显示，与类型颜色一致
